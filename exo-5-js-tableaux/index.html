Faire 2 tableaux : ( 12 Mois) : Janvier, Février, Mars ... ( 7 Jours) : Lundi, Mardi, Mercredi ... Faire de nouveau 2 tableaux : ( 12 Mois) : January, Febuary, March ... ( 7 jours) : Monday, Tuesday, Wednesday ... Vous devez stocker les 4 tableaux dans un autre tableau.

Cependant : Vous devez stocker les 2 tableaux français ensemble et les 2 autres tableaux anglais ensemble également. Dans la console, afficher successivement : Novembre Friday Tous les mois en Anglais , ex. "January, February, March, ... " Tous les jours en Francais ex. "Lundi, Mardi, Mercredi, ... "
